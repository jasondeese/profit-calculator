# Restaurant Profit Manager

A lightweight **React + Tailwind** app to track restaurant orders, costs, expenses, and profit â€” all stored locally in `localStorage`.

---

## Features

- Add, edit, or remove menu items (price, cost, optional inventory)  
- Build orders, adjust quantities, and place them  
- Track Revenue, COGS, Gross Profit, Expenses, and Net Profit  
- Daily summary of orders  
- Add expenses, export CSV, reset day, and load sample data  

---

## Installation

```bash
git clone https://github.com/yourusername/restaurant-profit-manager.git
cd restaurant-profit-manager
npm install
npm start
Open http://localhost:3000 in your browser.

Usage
Menu: Add or edit items

Orders: Add to cart, place orders

Expenses: Track daily expenses
